1. 内容总结

本文将指导你为 Hexo 博客创建 三种目录：

  1. 分类目录：按主题分类文章（如“技术教程”“生活随笔”）。
  2. 标签目录：按关键词标记文章（如“Hexo”“GitHub”）。
  3. 文章内目录：自动生成文章内部的导航目录（TOC）。

2. 分步操作指南

4. 常见问题

Q1：分类/标签页显示 404 怎么办？

  • 原因：未正确生成页面或配置错误。
  • 解决：
    1. 确保执行了 hexo new page categories 和 hexo new page tags。
    2. 检查生成的 index.md 文件中是否包含 type: "categories" 或 type: "tags"。

Q2：文章内目录未生成怎么办？

  • 原因：未启用 TOC 功能或未使用标题标记。
  • 解决：
    1. 确认文章中使用 ##、### 等标题标记。
    2. 检查主题配置中 toc.post 是否为 true。
